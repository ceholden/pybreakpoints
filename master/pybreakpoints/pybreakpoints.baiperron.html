pybreakpoints.baiperron module¶

Bai & Perron test for detecting multiple unknown breakpoints

pybreakpoints.baiperron.breakpoint(X, y, h=0.15, breaks=None)[source]¶

Bai & Perron breakpoint estimation

Parameters:
  • X (array-like, shape (n, p)) – Independent variables
  • y (array-like, shape (n, )) – Dependent variable
  • h (int or float, optional) – Minimum segment size. If float, assumed to be fraction of total number of observations
  • breaks (int, optional) – Number of breaks to fit. If None, will fit as many as possible given h minimum segment size
Returns:

  • dict[int, np.ndarray] – Locations of detected breaks for all possible numbers of breaks (1 - breaks)
  • np.ndarray – Total RSS for each number of fitted breaks
  • np.ndarray – Bayesian Information Criterion (BIC) for each number of fitted breaks (lower is better)
